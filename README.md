# Price Comparison Test: Flipkart vs Amazon

This repository contains a Playwright + TypeScript end‑to‑end test that compares the price of **iPhone 15 Plus** on Flipkart and Amazon.

## 🚀 Tech Stack

- Node.js (>=14)
- Playwright Test  
- TypeScript
